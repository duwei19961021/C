##### 整形在内存中的存储

- 原码 (无符号数或则正数原反补相同)

  数的二进制序列

  ```c
  char a = 10;
  // 0000 1010
  char b = -10;
  // 1000 1010
  ```

  

- 反码

  负数：原码的符号位不变，其它位取反。

  ```c
  char a = 10;
  // 0000 1010
  char b = -10;
  // 1111 0101
  ```

  

- 补码

  负数：反码+1

  ```c
  char a = 10;
  // 0000 1010
  char b = -10;
  // 1111 0110
  ```

```c
signed char; // -127~128
// 1000 0000 (-0) 规定为-128
// 0000 0000 (0)
```
